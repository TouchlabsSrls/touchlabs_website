# Purpose

Questo documento rappresenta il punto di partenza dell'intero progetto.

Prima di progettare anche una sola pagina del sito è fondamentale comprendere chi è Touchlabs.

L'obiettivo NON è realizzare un sito internet.

L'obiettivo è costruire un'identità digitale capace di rappresentare l'azienda nei prossimi anni.

Ogni decisione progettuale dovrà essere coerente con questa filosofia.

---

# Chi è Touchlabs

Touchlabs è una software house italiana specializzata nello sviluppo di soluzioni digitali ad alto valore aggiunto.

L'azienda progetta software custom, piattaforme web, sistemi gestionali, configuratori 3D, applicazioni immersive, soluzioni di Spatial Computing e strumenti basati sull'Intelligenza Artificiale.

Touchlabs non vende semplicemente software.

Progetta strumenti che aiutano le aziende a lavorare meglio, innovare più velocemente e creare nuove esperienze digitali.
