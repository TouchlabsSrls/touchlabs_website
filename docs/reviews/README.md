## Regole comuni

- Non inventare nuove sezioni
- Non modificare brand, palette o tipografia
- Non aumentare la quantità di contenuto (privilegiare tagli e accorpamenti)
- Distinguere bloccanti / importanti / opzionali
- Indicare pagina, sezione, selettore o file
- Non applicare modifiche finché il piano non è approvato
